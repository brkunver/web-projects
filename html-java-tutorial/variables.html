2. Variables in Java

We use variables to store our data
There are some basic data types :

  • String :
    We use String to store plain text
    for example : "Java"
    To use String variable first we write String then variable name
    -> String name
    Also we can assign it's value using "=" symbol
    String name = "name" ;

  • Integer :
    We use Integer to store whole number like 5 or 85
    To use Integer variable first we write int then variable name
    -> int name ;
    Also we can assign it's value using "=" symbol
    int name = 985 ;

  • Float :
    We use Float to store decimal number like 5.5 or 85.9
    To use Float variable first we write float then variable name
    -> float name ;
    Also we can assign it's value using "=" symbol
    float name = 9.98 ;

  • Boolean :
    We use Boolean to store true or false
    To use Boolean variable first we write boolean then variable name
    -> boolean name ;
    Also we can assign it's value using "=" symbol
    boolean name = true ;
    boolean name = false ;
